Approximate string matching

It is a technique of finding approximate matches to a pattern in a string

approximate matching is an update for the exact matching

the difference between the exact and the approximate matching is that:
the exact matching: does not allow for mistakes (mutations) when it finds a mistake it stop
but in the approximate matching it allows for mismatches (mutations) between the two sequences because in the real life there are mutations.
